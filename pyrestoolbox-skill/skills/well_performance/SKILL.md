# Well Performance Skill

Focused on productivity and production system optimization.

## When to Use
Invoke this skill for:
- **IPR Curves**: Vertical/horizontal wells, Vogel model for P < Pb.
- **VLP Curves**: Multiphase correlations (HB, WG, Gray, BB).
- **Nodal Analysis**: Finding the operating point of the well system.
- **Rate Calculation**: Radial/linear flow equations.

## Workflow
1. Define the Reservoir inflow (IPR) parameters.
2. Define the Tubing outflow (VLP) constraints.
3. Apply Nodal Analysis to find the intersection.
4. Plot the results with the specialized 'line' JSON output format.
